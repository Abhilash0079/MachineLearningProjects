# 📈 Sales Forecasting for Retail Stores (Time Series Machine Learning Project)

## 🧩 Project Overview
This project aims to forecast daily sales for a retail store using historical sales data. 
Accurate forecasting helps businesses optimize inventory, staff planning, and marketing budgets.

## 🎯 Business Objectives
- Predict future sales (next 30 days)
- Identify trends and seasonal patterns
- Improve inventory & supply chain decisions

## 📊 Dataset
The dataset contains:
- Date
- Daily Sales
- Store Information
- Holiday Indicators
- Promotions

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Linear Regression
- Lasso Regression
- XGBoost / Random Forest

## 🧠 Machine Learning Models Used
- Linear
- Lasso
- Random Forest Regressor / XGBoost

## 📂 Project Structure

## 📌 Workflow
1. Data Cleaning
2. Feature Engineering
3. Time-Series EDA
4. Train-Test Split (Time-based)
5. Model Training & Hyperparameter Tuning
6. Forecasting Future Sales
7. Evaluation (RMSE, MAPE, MAE)

## 📈 Results
- Trend: Increasing/Decreasing Trend
- Seasonality: Weekly and Monthly Patterns
- Best Model: XGBoost (lowest error)
- Forecast for next 30 days included
